# 🔵 Juego de Burbujas Interactivas - Canvas API

Un proyecto web interactivo desarrollado con la API de Canvas de HTML5. El objetivo del juego es reventar burbujas generadas aleatoriamente en la pantalla antes de que escapen por la parte superior, aumentando la dificultad progresivamente a través de niveles configurables.

---

## 🚀 Tecnologías Utilizadas

Este proyecto fue construido utilizando tecnologías web estándar sin frameworks de JavaScript complejos, priorizando el rendimiento y la manipulación directa del DOM y Canvas. Además, se utilizó Bootstrap para un diseño responsivo y estilizado con Glassmorphism.
